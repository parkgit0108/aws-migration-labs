# Create IAM Group

---

<aside>

**Purpose:** Create an **IAM user group** to manage permissions for multiple IAM users in one place.

</aside>

---

### Procedure

1. Sign in to the AWS Console using the provided credentials and URL.
2. Open **IAM**:
    1. In the AWS search bar, type **IAM**.
    2. Select **IAM** from the results.
3. In the left sidebar, go to **User groups**.
4. Click **Create group**.
    
    ![image.png](Create%20IAM%20Group/image.png)
    
5. Enter a **Group name** provided.
6. (Optional) Add existing users to the group.
7. Click **Create group**.
8. The user group now should appear in the list.
    
    ![image.png](Create%20IAM%20Group/image%201.png)
