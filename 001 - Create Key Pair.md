# Create EC2 Key Pair

<aside>

**Purpose:** Create an **EC2 key pair** in AWS so you can securely connect to an EC2 instance.

</aside>

---

### Procedure

1. Sign in to the AWS Console using the provided credentials and URL.
2. Open **Amazon EC2**.
3. In the left sidebar, under **Network & Security**, click **Key Pairs**.
4. Click **Create key pair**.
5. Fill out the form with the required information.
6. Click **Create key pair**.
7. When the key pair downloads automatically, store it somewhere secure.

### Quick check

- **Location:** EC2 → Network & Security → Key Pairs
- **Expected result:** Your new key pair appears in the list, and the private key file has been downloaded and stored safely
