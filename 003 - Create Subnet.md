1. Sign in using the user credential provided and use the URL given.
2. Search for VPC and go to the VPC dash board.
3. Navigate to Subnets and click create subnet.
4. Select the default VPC in the drop down.
5. fill out the form with information provided.
   NOTE - the IPv4 subnet CIDR block shouldn't overlap with the existing subnet. Use the next arrow button to select next available subnet.
