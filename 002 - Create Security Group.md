# Create Security Group

<aside>

**Purpose:** Create a new **security group** in AWS EC2 using the provided configuration.

</aside>

---

### Procedure

1. In the **EC2 Dashboard**, in the left sidebar under **Network & Security**, click **Security Groups**.
2. Click **Create security group**.
3. Fill out the form using the provided information:
    - **Security group name**
    - **Description**
    - **Inbound rules**
4. Click **Create security group**.

---

### Quick check

- **Location:** EC2 → Network & Security → Security Groups
- **Expected result:** The new security group appears in the list with the expected inbound rules
